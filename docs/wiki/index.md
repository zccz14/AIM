# AIM Wiki

本 wiki 是面向 LLM 的维护层，用来回答“这个仓库是干啥的”。原始资料仍以仓库中的 README、`docs/`、模块 README 与源码为准；这里仅做可追溯的摘要、综合判断与待确认记录。

## 阅读入口

- 想快速回答“这个仓库是干啥的”：见 [topic/what-this-repo-is-for.md](./topic/what-this-repo-is-for.md)
- 想看当前已落地能力与模块分布：见 [topic/baseline-and-module-map.md](./topic/baseline-and-module-map.md)
- 想看关键边界、非目标与避免误读点：见 [topic/boundaries-and-non-goals.md](./topic/boundaries-and-non-goals.md)
- 想看仍待确认事项：见 [maintenance/open-questions.md](./maintenance/open-questions.md)

## 使用约定

- **来源事实**：直接来自 raw sources 的可定位陈述。
- **综合判断**：基于多个来源整理出的解释，不等同于 raw source 原句。
- **待验证项**：当前资料不足，不能写成稳定事实的内容。

首次 ingest 记录见 [log.md](./log.md)。
